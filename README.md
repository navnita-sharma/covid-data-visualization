**Author:** Navnita Sharma

---

## 📊 Project Overview
This project explores global and country‑level COVID‑19 data using **RMarkdown** and **ggplot2**. The goal is to analyze trends, compare countries, and visualize correlations in a clear, interactive report.

---

## 🛠️ Tools & Libraries
- **RMarkdown** for reproducible reporting  
- **ggplot2** for data visualization  
- **dplyr** and **tidyr** for data wrangling  
- **readr** for dataset import  

---

## 📈 Visualizations Included
1. **Global Trend Line Chart** – confirmed cases over time  
2. **Top 10 Countries by Confirmed Cases** – horizontal bar chart  
3. **Deaths vs Recovered Scatter Plot** – country‑level comparison  
4. **Stacked Bar Chart (Selected Countries)** – breakdown for India, US, China, Canada, France  
5. **Correlation Plot (Deaths vs Confirmed)** – regression line showing relationship
